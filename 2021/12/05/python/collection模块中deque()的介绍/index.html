collection模块之deque()的介绍

2021年12月05日 697 字 大概 3 分钟

什么是collection模块？
collection模块提供了一些python常用的数据类型，为python内置的类型提供补充
常用deque,Counter,defaultdict类型

什么是deque?

A listlike sequence optimized for data accesses near its endpoints.

collection.deque 是在python标准库中（据说）最好用的双向队列类型，其他方法有queue.Queue / asyncio.Queue / multiprocessing.Queue

什么是双向队列？

1、双向队列是指一种具有队列和栈的性质的数据结构。
2、双向队列中的元素可以从两端弹出，其限定插入和删除操作在表的两端进行。

双向队列就像一个可固定长度的列表，它的长度固定后，在一边添加元素后，会在另一边删除删除元素。

collection.deque属性介绍

maxlen

maximum size of a deque or None if unbounded

即想要给该deque()队列设定的最大长度。

collection.deque类中方法介绍

下面对其类中定义的方法依次介绍：

append(…)

Add an element to the right side of the [deque]

在deque()对象右边添加一个元素，类似list类型。

appendleft(…)

Add an element to the left side of the [deque]

如上所示，在deque()对象左边添加一个元素。

clear()

Remove all elements from the [deque]

删除这个duque()对象中的所有元素。

copy(…)

Return a shallow copy of a [deque]

创建一个deque的 浅拷贝副本（浅拷贝，详见[[python中浅拷贝与深拷贝介绍]]）。

count(…)

return number of occurrences of value

返回deque()对象中元素的个数。

extend(…)

Extend the right side of the deque with elements from the iterable

在deque()对象右边连接可迭代元素，类似list类型。

extendleft(…)

Extend the left side of the deque with elements from the iterable

在左边 ……

index

return first index of value.
Raises ValueError if the value is not present.

在[start,[stop]]中返回所求元素的索引，若该元素不存在，触发ValueError错误。

insert(…)

  insert object before index

在目标对象的指定索引前添加对象。

pop(…)

Remove and return the rightmost element.

弹出并删除该deque()对象最右边的元素。

popleft(…)

还是从左边……

remove(…)

remove first occurrence of value.

删除第一次出现的指定元素（应该是左起？）
若没有指定元素，则返回ValueError。

reverse(…)

  reverse IN PLACE

翻转deque()对象中的元素，返回None。

retate(n=1,…)

Rotate the deque n steps to the right (default n=1).  
If n is negative, rotates left.

$n > 0$，向左旋转，即将最右边的元素翻到最左边。
$n < 0$，向右旋转，即将最左边的元素翻到最右边。
$n$为多少，就执行此操作多少次。
